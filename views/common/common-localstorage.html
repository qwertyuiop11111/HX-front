通用业务模块 - localStorage

localStorage功能

提供localStorage相关的增删改查方法。

字段值采用base64加密方式。

默认的有效时间段为30分钟。

简单样例
调用方式
             
                /*
                **存入localstorage的键值
                */
               var isSet=HXcommon.localStorage.setValue(key,value);
                 /*
                * 根据键名取出对应的值，第二个参数为可选项，设置逾期时间(秒)，默认为30分钟
                */
              var value=HXcommon.localStorage.getValue('name',3600);
                   /*
                * 清除某一个条目
                */
              var isClear=HXcommon.localStorage.clearValue('name');
