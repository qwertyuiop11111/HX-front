通用业务模块 - 表单与url

表单存取功能

提供表单数据转对象与将json数据赋值给表单的接口。

简单样例
调用方式
             
                /*
                **获取表单中所有元素值，键为name属性，返回为对象
                * formId:dom容器id
                */
              HXcommon.serializeJson('Form');
                 /*
                *将json对象赋值给form
                * formId:dom容器id  jsonValue:需要给form赋值的json对象
                */
              HXcommon.setForm('Form',jsonValue);
url参数获取功能

提供url所有参数与单个参数获取的接口。

简单样例
调用方式
             
                /*
                *取得URL参数、并传化成JSON
                * */
              HXcommon.getQueryJson();
                 /*
                *获取某个URL参数
                * */
              HXcommon.getUrlParam('taskid');
